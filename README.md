# smart-gate-system
Smart gate automation system using Arduino, ultrasonic sensor, and servo motor. Includes Blockly-based control logic and a web dashboard for gate status simulation.

## 🚀 Features
- Automatic gate opening using ultrasonic sensor
- Servo motor controlled gate system
- LED indication during gate movement
- Blockly-based custom control logic
- Web dashboard for gate status

## 🛠 Tech Used
- Arduino (C++)
- Ultrasonic Sensor (HC-SR04)
- Servo Motor
- Blockly (JavaScript)
- HTML, CSS, JavaScript

## 📂 Files
- sketch.ino → Arduino code
- index.html → Web dashboard
- blockly_code.txt → Blockly logic
